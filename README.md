# Virtual Memory Page Replacement Simulator

## Overview

This project is a **C++ based Virtual Memory Simulator** that demonstrates how operating systems manage memory using paging and page replacement algorithms.

The simulator models the behavior of **page frames in main memory** and analyzes how different page replacement strategies affect page faults and memory performance.

Users can provide a sequence of page references, and the simulator displays:

* Page hits and page faults
* Frame allocation after each memory access
* Total page faults generated during execution

This project is designed to help understand **virtual memory management**, a fundamental concept used in modern operating systems and processor architectures.

---

## Key Concepts Demonstrated

* Virtual Memory
* Paging
* Page Frames
* Page Hits and Page Faults
* Page Replacement Algorithms

---

## Features

* Interactive page reference input
* Visualization of frame allocation
* Page fault detection
* Simulation of page replacement strategies
* Memory access statistics

---

## Learning Objective

This simulator helps students and engineers understand how operating systems manage limited physical memory efficiently through paging mechanisms.

The concepts implemented in this project are widely used in modern computing systems designed by companies such as NVIDIA, Intel, and AMD.
